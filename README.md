# Learn Mindustry Mod

本仓库是Mindustry游戏mod开发教程的Github.io静态页面。

## 编写
使用IDEA/VSCODE直接编辑markdown，使用`pnpm dev`进行预览。
## 编写规范
- 每节标题使用Markdown的一级标题
- 每章标题写在各文件夹`index.md`文件中
- 图片使用相对路径引用
