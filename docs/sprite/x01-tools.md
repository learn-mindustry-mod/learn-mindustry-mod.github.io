# mindustry的像素艺术

## 工具获取

mindustry的贴图是一种不太入流的像素美术风格，笔者更倾向于称呼其为简几何风，是一种**精灵像素艺术(Sprite pixel art)**，学习贴图绘画的过程并不是孤立的，我们可以选择加入一些绘画社群，与其他贴图绘画爱好者交流和学习。这样不仅可以丰富绘画知识，还可以结识一些同道中人，共同进步。

### Pixel Studio [Pixel Studio](https://store.steamchina.com/app/1204050/Pixel_Studio__GIF/)

支持图层操作、逐帧动画制作(附带洋葱皮工具)，使用脚本绘制图片、并能将动画导出为GIF或精灵图。此外，它还提供调色板功能、跨平台同步功能、序列图功能、浏览作品功能、对称绘图功能和自定义画笔功能。

设备与平台: **Windows** **Linux** **iOS** **Android** **Steam**

拥有专业且全面的绘画功能，容易上手的UI设计，无缝的跨平台体验，等等一系列优秀功能的绘画工具，是所有人的不二之选，免费版就拥有几乎所有功能，PRO版只需要贪吃蛇小游戏通关即可获得，甚至还有自己的能保存全部信息的图片格式，最推荐这款软件。

- 许多用户长期使用；
- UI内容较多，方便进行繁杂绘画；
- UI便于使用，适合初学者；
- 可以使用脚本。

### Aseprite [Aseprite](https://www.aseprite.org/)

设备与平台: **Windows** **MacOS** **Linux**

功能全面，发展时间较长的像素艺术与精灵动画制作软件，但这款软件的UI不太适合mindustry的艺术创作形式。

- 许多用户长期使用；
- UI内容较多，方便进行繁杂绘画；
- 可以使用脚本。

## 工具配置与使用

下文讲述的是**Windows**上的**PS**的基础配置与使用方法，一般使用简单图层模式。

在**菜单**中新建、保存、打开一张画布。
在**图层**中的设置来进行自动对称。
在**功能**中进行手动对称、翻转、复制、黏贴、克隆或获取色板。
使用**范围工具**来选中范围内的画布进行操作。
使用**魔棒工具**来选中附近的同种颜色进行操作。
使用**框选工具**来选中一个矩形内的画布进行操作。
使用**渐变工具**进行颜色的渐变操作。
使用**形状工具**创建几何形状。
使用**触控工具**切换绘画模式。
