# 前言

从Mindustry更改单位艺术画风到现在，我们在美术方面有可观的进展，但人与人之间的差距巨大，各种"结构""构型"等等术语混乱而又不明确，贴图风格千
奇百怪而又参差不齐。

本篇教程旨在为贴图初学者提供基本的绘画规则，提供一些软件/思路来帮助贴图绘画，通读教程后的初学者至少可以画出能入眼的贴图作品。

在你阅读本教程之前，你至少需要有如下的知识与素养：

- 逐字阅读文章并吸收的能力；
- **能够长期自我练习**

最终，你将有能力轻松绘制贴图；创造自己的画风；甚至教会别人如何绘画贴图！当然，我说的只是美术层面上。在这个过程中，必不可少的就是练习和理解，这两项非常重要。
