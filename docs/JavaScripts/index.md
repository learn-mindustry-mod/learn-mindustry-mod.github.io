# 前言

虽然Mindustry的js充满争议,主流的模组也很少用js编写.但既然你打开了这篇教程,说明你对js模组开发有兴趣,甚至想要成为一名js modder.

本教程旨在为js模组初学者提供基本的开发理路,提供一些模板来帮助模组开发,最重要的是,在过程中学会阅读Java程序,通过查阅源代码达到自己想要的效果.

在你阅读本教程之前,你至少需要有如下的知识与素养：

- 基本的算术知识；
- 基本的编程知识,至少要知道什么是变量、函数、对象等概念；
- 逐字阅读文章并吸收的能力；
- **一颗常在学习的心**
